# Final Demo Script

1. Select ticker (`ASELS` sample).
2. Submit KAP ingest job and wait for completion.
3. Submit news ingest job and wait for completion.
4. Submit report ingest job (PDF) and wait for completion.
5. Ask consistency question in `Query` panel.
6. Show citations in response and `as_of_date`.
7. Show disclaimer string:
   - `This system does not provide investment advice.`
8. Open `Advanced Insight` output:
   - tension index
   - source mix
   - retrieval trace
9. Run evaluation (`hybrid`) and present:
   - rubric scores
   - latest report artifact path
